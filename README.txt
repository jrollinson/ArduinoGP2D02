This is for interfacing with Sharp GP2D02 infrared sensors with an arduino. 

TO INSTALL:
	If you haven't already, create a directory called 'libraries' in your sketchbook directory. Copy the GP2D02 folder into that directory.

This software is free to use in your own software, but please do not steal it as your own.  Simply acknowledge me.  Thanks a lot.
